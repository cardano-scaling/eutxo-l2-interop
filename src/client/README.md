# HTLC & Vesting Client

Next.js client application for interacting with Hydra heads, HTLC and Vesting contracts.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Key Features

- **Hydra Head Selection**: Browse and select Hydra heads
- **HTLC Management**: Create, claim, and refund HTLC transactions
- **Vesting Management**: Claim vesting contracts after timeout
- **UTXO List**: View HTLC, Vesting, and user UTxOs
- **User Switching**: Switch between Alice, Bob, and Ida users
